# 使用說明

1.  將同一筆資料的數個.dat檔一一拖曳至檔案上傳方框
2.  在輸入欄打入該筆資料的編號
3.  在輸入欄按下Enter，表格便會新增該筆資料欄位的計算結果
4.  可以重複前述步驟建立多筆資料，也可以點選每筆資料最左邊的按鈕以刪除一筆資料
5.  完成後，點選「複製表格」按鈕，即可至Excel貼上
